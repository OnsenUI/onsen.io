<%- @markdown => %> ## Onsen UIとVue 2でハイブリッドアプリやモバイルWebアプリを開発しよう Onsen UIをVue 2で使いやすくするためのバインディングを作成しました。Onsen UIは **Web Components** に準拠した、ハイブリッドアプリ開発のためのライブラリです。 ## NPMからダウンロード このバインディングは [NPMにて公開](https://www.npmjs.com/package/vue-onsenui)されています。利用するためには、Onsen UI コアライブラリも必要です。 アプリに組み込むには、下記のように yarn コマンドを用いてプロジェクトに加えてください。 ```bash yarn add onsenui vue-onsenui --save # npm コマンドでも可 ``` また、Unpkg CDN から直接 [コアライブラリ](https://unpkg.com/onsenui/) と [バインディング](https://unpkg.com/vue-onsenui/) をダウンロードすることもできます。 ## 簡単に利用できる ライブラリはVueプラグインとして配布されているため、下記のように簡単にアプリで使用できます。 ```javascript /** * スタイルシートは webpack または * link タグで読み込みます */ import 'onsenui/css/onsenui.css'; import 'onsenui/css/onsen-css-components.css'; // Vue をインポート import Vue from 'vue'; // Onsen UI コアライブラリと Vue バインディングをインポート import 'onsenui'; import VueOnsen from 'vue-onsenui'; // Vue バインディングをプラグインとして適用 Vue.use(VueOnsen); new Vue({ el: '#app', template: '', components: { App } }); ``` これにより Onsen UI が持つ全てのカスタム要素が `` というコンポーネント名で使えるようになります。また、プラットフォーム別処理や簡易アラートダイアログ表示を行えるユーティリティ関数も使えるようになります。 ## 詳細を知るには ライブラリの使い方は、下記のドキュメントに記述しています。また、多くのOnsen UIコンポーネントを組み込んだサンプルアプリも提供しています。 * [ドキュメント](/v2/docs/guide/vue/) * [GitHubレポジトリ](https://github.com/OnsenUI/OnsenUI/tree/vue2/bindings/vue) * [サンプルコード](https://github.com/OnsenUI/OnsenUI/tree/vue2/bindings/vue/examples/components) このライブラリは α 版の段階ですが、今後はドキュメントを増やしたり、機能を追加していきます。Vue 2を使ったモバイルアプリ開発を強力にサポートしていきます。 <% end %>
