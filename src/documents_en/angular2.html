Common core for AngularJS, Angular 2 and other frameworks

Onsen UI is compatible with both AngularJS and Angular 2 by sharing the same JavaScript and CSS core. It even supports other libraries like React and jQuery. Thus, you can easily switch between the libraries and frameworks and still feel at home.
