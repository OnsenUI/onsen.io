<%- @markdown => %> ## Create awesome mobile apps with Vue and Onsen UI Onsen UI is a Web Components based cross-platform mobile UI framework and toolset utilizing Cordova / PhoneGap. Onsen provides the complete mobile stack that enables web developers to build mobile apps for all major app stores with a single code base. It is super flexible and works with or without any JaveScript frameworks. For tighter integration, we provide "bindings" for major JS frameworks. With the Vue bindings, Onsen UI provides you with Vue 2 components and directives that wrap the core Web Components and expose a Vue-like API to interact with them. Vue.js developers, you can now create beautiful and high-performance mobile apps in no time in the most familiar way. ## Installing Vue bindings for Onsen UI The bindings are [available on NPM](https://www.npmjs.com/package/vue-onsenui). To use them you also need the core Onsen UI library. Therefore, to install the library in your app all you need to do is: ```bash yarn add onsenui vue-onsenui # Or use npm ``` It is also available from Unpkg CDN: [core](https://unpkg.com/onsenui/) and [bindings](https://unpkg.com/vue-onsenui/). ## Easy to use The library is distributed as a Vue plugin so you can easily add it to your application: ```javascript /** * The stylesheets can be loaded with Webpack * or added with link tags. */ import 'onsenui/css/onsenui.css'; import 'onsenui/css/onsen-css-components.css'; // Import Vue import Vue from 'vue'; // Import Onsen UI core and Vue bindings import 'onsenui'; import VueOnsen from 'vue-onsenui'; // And just plug the bindings Vue.use(VueOnsen); new Vue({ el: '#app', template: '', components: { App } }); ``` This exposes all the Onsen UI custom elements as `` components as well as some utility functions for platform recognition and creating quick alert dialogs. ## Learn more * [Docs](/v2/docs/guide/vue/) * [GitHub repository](https://github.com/OnsenUI/OnsenUI/tree/master/bindings/vue) * [Examples](https://github.com/OnsenUI/OnsenUI/tree/master/bindings/vue/examples/components) This library is still an beta version, but all components are implemented and fully working. We'd like to get your feedback to improve the API, so everyone can flawlessly start building mobile apps with Vue and Onsen UI! <% end %>
