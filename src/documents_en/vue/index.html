<%- @markdown => %> ## Create hybrid mobile apps with Onsen UI and Vue.js Integrating Onsen UI with Vue.js is now really simple. Vue bindings for Onsen UI provides a Vue-like API to interact with Onsen UI elements in the shape of Vue components and directives. All the power of Onsen UI is now available for your Vue.js apps. ## Installing Vue bindings for Onsen UI The bindings are [available on NPM](https://www.npmjs.com/package/vue-onsenui). To use them you also need the core Onsen UI library. Therefore, to install the library in your app all you need to do is: ```bash yarn add onsenui vue-onsenui --save # Or use npm ``` It is also available from Unpkg CDN: [core](https://unpkg.com/onsenui/) and [bindings](https://unpkg.com/vue-onsenui/). ## Easy to use The library is distributed as a Vue plugin so you can easily add it to your application: ```javascript /** * The stylesheets can be loaded with Webpack * or added with link tags. */ import 'onsenui/css/onsenui.css'; import 'onsenui/css/onsen-css-components.css'; // Import Vue import Vue from 'vue'; // Import Onsen UI core and Vue bindings import 'onsenui'; import VueOnsen from 'vue-onsenui'; // And just plug the bindings Vue.use(VueOnsen); new Vue({ el: '#app', template: '', components: { App } }); ``` This exposes all the Onsen UI custom elements as `` components as well as some utility functions for platform recognition and creating quick alert dialogs. ## Learn more * [Docs](/v2/docs/guide/vue/) * [GitHub repository](https://github.com/OnsenUI/OnsenUI/tree/vue2/bindings/vue) * [Examples](https://github.com/OnsenUI/OnsenUI/tree/vue2/bindings/vue/examples/components) This library is still an early preview but we are planning to add more features and also publish more documentation so everyone can start building mobile apps with Vue and Onsen UI! <% end %>
